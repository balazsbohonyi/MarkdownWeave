# Glossary

## Attention score

An attention score is the unnormalized compatibility value between a query vector and a key vector, commonly written as $q_i^\top k_j / \sqrt{d_k}$.

## Head

A head is one learned projection path in multi-head attention. Multiple heads can emphasize different relationships such as syntax, discourse role, or positional offset.

## Residual stream

The residual stream is the vector sequence passed through stacked layers while adding transformed outputs back to the current representation.
